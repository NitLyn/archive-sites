About NitLyn's File Compressor

Nitlyn is a lossless compression application that allows its user to compress files without changes in quality of pictures or videos using their lossless compression alogrithm. Nitlyn is a multi platform based on computer only (for the time being). NitLyn's FC allows its users send files directly to Discord, because of Discords 8.00mb Limit. No Need For Nitro!

NitLyn Runs On

  • Sveltejs
  • Neutralinojs
  • GraphQL
  • MySQL
  • MongoDB
